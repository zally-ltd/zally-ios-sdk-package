# Zally iOS SDK (binary package)

Public binary-only distribution of the Zally iOS SDK.

## Installation

Add to your app's `Package.swift` or in Xcode: File → Add Package Dependencies:

```
https://github.com/zally-ltd/zally-ios-sdk-package
```

Use a version (e.g. `1.0.0`) or branch. Then:

```swift
import zally_sdk
```

## First-time setup

See the **private** repo's **START_FROM_SCRATCH.md** for creating this repo and the first release. For how releases work (manual publish, no deploy on merge, hidden commit messages), see **docs/PUBLISHING.md** in the private repo.
